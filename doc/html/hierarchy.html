Hiérarchie des classes
Cette liste d'héritage est classée approximativement par ordre alphabétique :
[Niveau de détails 12]
 CConfigurationStructure permettant de configurer une communication
 CEtatManetteBoutonStructure qui définit l'état de la manette en mode pilotage de la pince
 CEtatManetteDeplacementStructure qui définit l'état de la manette en mode déplacement du robot
 CEtatManettePilotageStructure qui définit l'état de la manette en mode pilotage de la pince
 CMesureStructure permettant de définir les propriété d'une mesure prise à une heure précise
 CPhotoStructure contenant les informations d'une photo de campagne
 ▼CQDialogLa classe QDialog est la classe de base des fenêtres de dialogue
 CIHMCreationCampagneClass permettant de créer une nouvelle campagne
 ▼CQGamepadLa classe QGamepad est utilisée pour accéder à l'état actuel du matériel de la manette de jeu connecté à un système
 CManetteClasse permettant une communication entre le rov et la manette
 ▼CQObjectLa classe QObject est la classe de base de tous les objets Qt. Elle permet à ces objets Qt de disposer entre autres du mécanisme de communication signal/slot
 CBaseDeDonneesClass permettant de s'interfacer avec la base de données
 CCampagneClass contenant les informations de la campagne en cours
 CCapteursClasse contenant les dernières informations issues des capteurs du rov
 CCommunicationRovClass permettant de mettre en place une communication avec le rov
 CRovClasse controlant tout les traitements en provenance et en direction de la communication avec le rov
 ▼CQThreadLa classe QThread fournit un moyen indépendant de gérer les threads dans Qt
 CCameraClass permettant de mettre en place une communication avec la camera
 ▼CQWidgetLa classe QWidget est la classe de base de tous les objets graphiques d'interface utilisateur
 CIHMAccueilClass permettant de créer une nouvelle campagne, reprendre une campagne mise en pause, archiver une campagne, supprimer une campagne, accéder à la base de données et configurer le matériel
 CIHMAlbumPhotoClass permettant de visualiser les photos en cours de campagne
 CIHMConfigurationClass permettant de configurer la communication avec le rov
 CIHMReglageVideoClasse permettant de regler l'affichage du flux video
 CIHMRovIHM permettant d'obtenir le flux vidéo en direct placé sur le robot et d'obtenir les informations relatifs à ses capteurs
